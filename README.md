# 📈 FAANG Stock Market Time Series Analysis

## 📌 Project Overview
This project performs Time Series Analysis on FAANG stock market data to identify trends, analyze technical indicators, detect volatility, and perform basic forecasting.

---

## 🎯 Objective
The objectives of this project are:
- Analyze stock price trends over time
- Study technical indicators
- Detect volatility and market patterns
- Perform basic stock forecasting

---

## 📂 Dataset
Dataset used:
- FAANG Stock Market Dataset with Technical Indicators

Features include:
- Open, High, Low, Close prices
- Volume
- Moving Averages (SMA, EMA)
- RSI
- MACD
- Volatility
- Daily Returns

---

## 🛠 Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook / Kaggle Notebook

---

## 📊 Analysis Performed

### ✔ Trend Analysis
- Visualized stock closing price trends over time

### ✔ Moving Average Analysis
- Compared SMA 7 and SMA 21 to identify trend direction

### ✔ RSI Analysis
- Detected overbought and oversold market conditions

### ✔ MACD Analysis
- Analyzed momentum and trend reversals

### ✔ Volatility Analysis
- Examined market fluctuations using 7-day volatility

### ✔ Forecasting
- Used Linear Regression for basic stock price prediction

---

## 🔍 Key Insights
- Apple stock showed strong long-term growth
- Moving averages highlighted trend consistency
- RSI identified periods of market pressure
- MACD captured momentum changes effectively
- Volatility spikes reflected uncertain market conditions

---

## 🧠 Conclusion
This project demonstrates how Time Series Analysis and technical indicators can be used to understand stock market behavior, identify trends, and support investment decision-making.
